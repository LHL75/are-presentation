# Modèle de Lotka-Volterra

### Les équations

Les équations du modèle de Lotka-Volterra sont données par :

$$
\begin{aligned}
    \begin{cases}
        u'(t) = a u(t) - b v(t) u(t) \\
        v'(t) = -c v(t)  + d u(t) v(t)
    \end{cases}
\end{aligned}
$$

où a, b, c et d sont des réels positifs.
