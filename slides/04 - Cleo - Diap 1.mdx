# Modèle de Lotka-Volterra

## Interprétation intuitive

### Les équations

Les équations du modèle de Lotka-Volterra sont données par :

$$
\begin{aligned}
    \begin{cases}
        u'(t) = a u(t) - b v(t) u(t) \\
        v'(t) = -c v(t)  + d u(t) v(t)
    \end{cases}
\end{aligned}
$$

où a, b, c et d sont des réels positifs.


### Dynamique du modèle

Le modèle de Lotka-Volterra décrit l'interaction entre deux espèces, les proies et les prédateurs, et montre comment leurs populations évoluent au fil du temps.

# Gaussienne avec Flèches de Concavité et Convexité

Voici une gaussienne avec des flèches indiquant la concavité et la convexité :

$$
\begin{tikzpicture}
    \begin{axis}[
        axis lines=middle,
        xlabel=$x$,
        ylabel=$y$,
        ymin=0, ymax=1.2,
        xmin=-3, xmax=3,
        width=10cm,
        height=6cm,
        grid=major,
        grid style={dashed}
    ]
        % Plot the Gaussian function
        \addplot[
            domain=-3:3,
            samples=100,
            thick,
            blue,
            postaction={decorate},
            decoration={
                markings,
                mark=between positions 0.1 and 0.9 step 0.1 with {
                    \arrow{latex}
                }
            }
        ] {exp(-x^2)};

        % Add arrows for concavity
        \addplot[
            domain=-3:-1,
            samples=10,
            thick,
            red,
            postaction={decorate},
            decoration={
                markings,
                mark=between positions 0.2 and 0.8 step 0.2 with {
                    \arrow[rotate=90]{latex}
                }
            }
        ] {exp(-x^2)};

        \addplot[
            domain=1:3,
            samples=10,
            thick,
            red,
            postaction={decorate},
            decoration={
                markings,
                mark=between positions 0.2 and 0.8 step 0.2 with {
                    \arrow[rotate=90]{latex}
                }
            }
        ] {exp(-x^2)};

        % Add arrows for convexity
        \addplot[
            domain=-1:1,
            samples=10,
            thick,
            green,
            postaction={decorate},
            decoration={
                markings,
                mark=between positions 0.2 and 0.8 step 0.2 with {
                    \arrow[rotate=-90]{latex}
                }
            }
        ] {exp(-x^2)};
    \end{axis}
\end{tikzpicture}
$$

Ce code LaTeX trace une gaussienne avec des flèches indiquant la concavité et la convexité. Les flèches sont orientées vers le bas dans les zones concaves et vers le haut dans les zones convexes.
